express-boilerplate
===================

A modern Express boilerplate for building RESTful APIs.

- Babel (transpiles to Node 4)
- ES2016, async/await, etc.
- Flow
- ESLint
- Mongoose
- Jest

## Install

```
yarn
cp .env.test .env
```

## Start dev server

```
yarn start
```

## Build for production

```
yarn build
```

## Run tests

```
yarn test
```
