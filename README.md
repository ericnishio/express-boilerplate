express-boilerplate
===================

A modern Express boilerplate for building RESTful APIs.

- Babel (transpiles to Node 4)
- ES2015+ (incl. async/await)
- Flow
- ESLint

## Install

```
yarn
cp .env.example .env
```

## Start dev server

```
yarn start
```

## Build for production

```
yarn build
```

## Run tests

```
yarn test
```
