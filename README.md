express-boilerplate
===================

A modern Express boilerplate for building RESTful APIs.

- Babel (transpiles to Node 6)
- ES2016, async/await, etc.
- Flow
- ESLint
- Mongoose
- Jest

## Install

```
yarn
cp .env.dev .env
```

## Start dev server

(Run the tests to lift up the database.)

```
yarn start
```

## Build for production

```
yarn build
```

## Run tests

```
./run-tests.sh
```
